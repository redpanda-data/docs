Learn how to build a chat room application in Go, Java, Node.js, or Python using a remote Redpanda cluster running in Redpanda Cloud.

- [Golang](../guide-go-cloud)

  Create a basic chat room application with Redpanda Cloud and Kafka clients developed with [franz-go](https://github.com/twmb/franz-go).

- [Java](../guide-java-cloud)

  Create a basic chat room application with Redpanda Cloud and Kafka Java clients.

- [Node.js](../guide-nodejs-cloud)

  Create a basic chat room application with Redpanda Cloud and Kafka clients developed with [kafkajs](https://kafka.js.org/).

- [Python](../guide-python-cloud)

  Create a basic chat room application with Redpanda Cloud and Kafka clients developed with [kafka-python](https://kafka-python.readthedocs.io/en/master/).
