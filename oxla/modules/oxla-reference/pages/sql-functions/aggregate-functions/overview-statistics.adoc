= Overview


Aggregate functions for statistics are typically used for statistical analysis. Oxla supports the following functions:

[cols="1,1",options="header"]
|===
|**Functions** |**Description**

|xref:oxla-reference:sql-functions/aggregate-functions/corr.adoc[CORR] |Calculates the Pearson correlation coefficient between two sets of number pairs
|xref:oxla-reference:sql-functions/aggregate-functions/covar-pop.adoc[COVAR\_POP] |Calculates the population covariance between two sets of number pairs
|xref:oxla-reference:sql-functions/aggregate-functions/covar-samp.adoc[COVAR\_SAMP] |Calculates the sample covariance between two sets of number pairs
|xref:oxla-reference:sql-functions/aggregate-functions/regr-avgx.adoc[REGR\_AVGX] |Calculates the average of the independent variable (sum(X)/N)
|xref:oxla-reference:sql-functions/aggregate-functions/regr-avgy.adoc[REGR\_AVGY] |Calculates the average of the dependent variable (sum(Y)/N)
|xref:oxla-reference:sql-functions/aggregate-functions/regr-count.adoc[REGR\_COUNT] |Calculates the number of input rows in which both expressions are non-null
|xref:oxla-reference:sql-functions/aggregate-functions/regr-intercept.adoc[REGR\_INTERCEPT] |Calculates the y-intercept of the univariate linear regression line for a group of data points
|xref:oxla-reference:sql-functions/aggregate-functions/regr-r2.adoc[REGR\_R2] |Calculates the coefficient of determination (R<sup>2</sup>) for a linear regression model
|xref:oxla-reference:sql-functions/aggregate-functions/regr-slope.adoc[REGR\_SLOPE] |Calculates slope of the least-squares-fit linear equation determined by the (X, Y) pairs
|xref:oxla-reference:sql-functions/aggregate-functions/regr-sxx.adoc[REGR\_SXX] |Calculates the sum(X<sup>2</sup>) - sum(X)<sup>2</sup>/N ("sum of squares" of the independent variable)
|xref:oxla-reference:sql-functions/aggregate-functions/regr-sxy.adoc[REGR\_SXY] |Calculates the sum(X<sup>*</sup>Y) - sum(X) * sum(Y)/N ("sum of products" of independent times dependent variable)
|xref:oxla-reference:sql-functions/aggregate-functions/regr-syy.adoc[REGR\_SYY] |Calculates the sum(Y<sup>2</sup>) - sum(Y)<sup>2</sup>/N ("sum of squares" of the dependent variable)
|xref:oxla-reference:sql-functions/aggregate-functions/stddev.adoc[STDDEV] |Calculates the sample standard deviation of a set of numeric values
|xref:oxla-reference:sql-functions/aggregate-functions/stddev-pop.adoc[STDDEV\_POP] |Calculates the population standard deviation of the input values
|xref:oxla-reference:sql-functions/aggregate-functions/stddev-samp.adoc[STDDEV\_SAMP] |Calculates the sample standard deviation of the input values
|xref:oxla-reference:sql-functions/aggregate-functions/variance.adoc[VARIANCE] |Calculates the the sample variance of a set of numeric values.
|xref:oxla-reference:sql-functions/aggregate-functions/var-pop.adoc[VAR\_POP] |Calculates the population variance of the input values (square of the population standard deviation)
|xref:oxla-reference:sql-functions/aggregate-functions/var-samp.adoc[VAR\_SAMP] |Calculates the sample variance of the input values (square of the sample standard deviation)
|===
