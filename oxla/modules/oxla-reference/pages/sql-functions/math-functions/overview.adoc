= Overview


Mathematical functions and Operators in Oxla are designed to perform mathematical calculations and manipulate integer or floating-point numbers. Oxla supports the following math functions:

[cols="1,1",options="header"]
|===
|**Function** |**Description**

|xref:oxla-reference:sql-functions/math-functions/abs.adoc[ABS()] |This function returns the absolute value of an argument, regardless of whether it is positive or negative
|xref:oxla-reference:sql-functions/math-functions/cbrt.adoc[CBRT()] |This function returns the cube root of a given number
|xref:oxla-reference:sql-functions/math-functions/ceil.adoc[CEIL()] |This function rounds up to the nearest positive or negative integer value greater than or equal to the argument
|xref:oxla-reference:sql-functions/math-functions/exp.adoc[EXP()] |This function returns the exponential value of a number specified in the argument
|xref:oxla-reference:sql-functions/math-functions/floor.adoc[FLOOR()] |This function returns a number rounded down that is less than or equal to the specified argument&#x20;
|xref:oxla-reference:sql-functions/math-functions/greatest.adoc[GREATEST()] |This function extracts the greatest or largest value from a set of values.
|xref:oxla-reference:sql-functions/math-functions/least.adoc[LEAST()] |This function returns the least or smallest value in a list of values&#x20;
|xref:oxla-reference:sql-functions/math-functions/ln.adoc[LN()] |This function returns the exponential value of its argument
|xref:oxla-reference:sql-functions/math-functions/log.adoc[LOG()] |This function returns the base-10 logarithm or logarithm of the specified base of a given number
|xref:oxla-reference:sql-functions/math-functions/power.adoc[POWER()] |This function returns the value of a number raised to the power of another number specified in the arguments
|xref:oxla-reference:sql-functions/math-functions/random.adoc[RANDOM()] |This function returns a random number between 0 (inclusive) and 1 (exclusive)
|xref:oxla-reference:sql-functions/math-functions/round.adoc[ROUND()] |This function rounds numbers to the nearest integer or to a specified number of decimal places
|xref:oxla-reference:sql-functions/math-functions/sign.adoc[SIGN()] |This function returns -1 for negative arguments, 1 for positive arguments or 0 if the argument is 0
|xref:oxla-reference:sql-functions/math-functions/sin.adoc[SIN()] |This function returns the trigonometric sine value of a specified angle in radians
|xref:oxla-reference:sql-functions/math-functions/sqrt.adoc[SQRT()] |This function returns the square root of its argument
|xref:oxla-reference:sql-functions/math-functions/bitwise-shift-left.adoc[BITWISE SHIFT LEFT] |This operator manipulate the bits of integer values by shifting them left
|xref:oxla-reference:sql-functions/math-functions/bitwise-shift-right.adoc[BITWISE SHIFT RIGHT] |This operator manipulate the bits of integer values by shifting them right
|xref:oxla-reference:sql-functions/math-functions/to-char-from-number.adoc[TO\_CHAR() from Number] |Formats a number into a string using a given format
|===
