= Window Functions

Window functions perform calculations across rows related to the current row, enabling advanced analytics like running totals, rankings, and moving averages without collapsing result sets.

== Aggregate Window Functions

Calculate values across a window of rows while preserving individual row identity:

* **xref:oxla-reference:sql-functions/window-functions/avg.adoc[AVG]** - Computes the average value across the window
* **xref:oxla-reference:sql-functions/window-functions/count.adoc[COUNT]** - Counts rows within the window
* **xref:oxla-reference:sql-functions/window-functions/sum.adoc[SUM]** - Totals numeric values in the window
* **xref:oxla-reference:sql-functions/window-functions/min.adoc[MIN]** - Finds the minimum value in the window
* **xref:oxla-reference:sql-functions/window-functions/max.adoc[MAX]** - Finds the maximum value in the window
* **xref:oxla-reference:sql-functions/window-functions/bool-and.adoc[BOOL_AND]** - Returns true if all values are true
* **xref:oxla-reference:sql-functions/window-functions/bool-or.adoc[BOOL_OR]** - Returns true if any value is true

== Ranking Functions

Assign rankings and positions to rows based on specified ordering:

* **xref:oxla-reference:sql-functions/window-functions/row-number.adoc[ROW_NUMBER]** - Assigns sequential numbers to rows
* **xref:oxla-reference:sql-functions/window-functions/rank.adoc[RANK]** - Assigns rank with gaps for ties
* **xref:oxla-reference:sql-functions/window-functions/dense-rank.adoc[DENSE_RANK]** - Assigns rank without gaps for ties
* **xref:oxla-reference:sql-functions/window-functions/percent-rank.adoc[PERCENT_RANK]** - Calculates relative rank as a percentage
* **xref:oxla-reference:sql-functions/window-functions/cume-dist.adoc[CUME_DIST]** - Computes cumulative distribution
* **xref:oxla-reference:sql-functions/window-functions/ntile.adoc[NTILE]** - Divides rows into specified number of groups

== Value Functions

Access values from specific positions within the window:

* **xref:oxla-reference:sql-functions/window-functions/first-value.adoc[FIRST_VALUE]** - Returns the first value in the window
* **xref:oxla-reference:sql-functions/window-functions/last-value.adoc[LAST_VALUE]** - Returns the last value in the window
* **xref:oxla-reference:sql-functions/window-functions/nth-value.adoc[NTH_VALUE]** - Returns the nth value in the window
* **xref:oxla-reference:sql-functions/window-functions/lag.adoc[LAG]** - Accesses data from a previous row
* **xref:oxla-reference:sql-functions/window-functions/lead.adoc[LEAD]** - Accesses data from a following row

Window functions use the `OVER` clause to define partitioning and ordering. They are essential for time-series analysis, ranking, running calculations, and comparative analytics.
