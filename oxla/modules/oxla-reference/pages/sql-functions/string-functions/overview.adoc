= String Functions

String functions in Oxla enable manipulation and analysis of text data through operations like concatenation, case conversion, pattern matching, and substring extraction.

== String Manipulation

* **xref:oxla-reference:sql-functions/string-functions/concat.adoc[CONCAT]** - Combines multiple strings into one
* **xref:oxla-reference:sql-functions/string-functions/replace.adoc[REPLACE]** - Substitutes occurrences of a substring with another string
* **xref:oxla-reference:sql-functions/string-functions/substr.adoc[SUBSTR]** - Extracts a portion of a string by position
* **xref:oxla-reference:sql-functions/string-functions/substring.adoc[SUBSTRING]** - Returns a substring based on position and length
* **xref:oxla-reference:sql-functions/string-functions/upper.adoc[UPPER]** - Converts all characters to uppercase
* **xref:oxla-reference:sql-functions/string-functions/lower.adoc[LOWER]** - Converts all characters to lowercase

== String Analysis

* **xref:oxla-reference:sql-functions/string-functions/length.adoc[LENGTH]** - Returns the number of characters in a string
* **xref:oxla-reference:sql-functions/string-functions/position.adoc[POSITION]** - Finds the location of a substring within a string
* **xref:oxla-reference:sql-functions/string-functions/strpos.adoc[STRPOS]** - Returns the position of a substring (alias for POSITION)
* **xref:oxla-reference:sql-functions/string-functions/starts-with.adoc[STARTS_WITH]** - Checks if a string begins with a specific prefix
* **xref:oxla-reference:sql-functions/string-functions/ends-with.adoc[ENDS_WITH]** - Checks if a string ends with a specific suffix

== Regular Expressions

* **xref:oxla-reference:sql-functions/string-functions/regex/posix-regular-expressions.adoc[POSIX Regular Expressions]** - Pattern matching syntax reference
* **xref:oxla-reference:sql-functions/string-functions/regex/regexp-match.adoc[REGEXP_MATCH]** - Extracts substrings matching a regex pattern
* **xref:oxla-reference:sql-functions/string-functions/regex/regexp-replace.adoc[REGEXP_REPLACE]** - Replaces text matching a regex pattern

All string functions are PostgreSQL-compatible and support standard SQL text operations for data processing and transformation.
