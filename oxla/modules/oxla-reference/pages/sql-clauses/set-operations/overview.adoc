= Overview


Set operations are operations used to manipulate and analyze sets. It includes the following operations:

1.  xref:oxla-reference:sql-clauses/set-operations/union.adoc[**Union**]: Combines two or more sets to create a new set containing all unique elements from the input sets.
2.  xref:oxla-reference:sql-clauses/set-operations/intersect.adoc[**Intersect**]: Yields a new set with elements common to all input sets.
3.  xref:oxla-reference:sql-clauses/set-operations/except.adoc[**Except**]: Generates a set containing elements from the first set that are not present in the second set.

These operations allow for comparisons, combinations, and distinctions among sets in various contexts.
