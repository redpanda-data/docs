= Deploy on Kubernetes
:description: Kubernetes is a container orchestration tool that helps you to manage Redpanda cluster deployments using declarative configuration files called manifests. Kubernetes provides a standardized way of achieving high availability, disaster recovery, and scalability.
:page-layout: index

Kubernetes is a container orchestration tool that helps you to manage Redpanda cluster deployments using declarative configuration files called _manifests_. Kubernetes provides a standardized way of achieving high availability, disaster recovery, and scalability.
