[IMPORTANT]
====
Redpanda's Linux package repositories are moving from `dl.redpanda.com` to `linux.pkg.redpanda.com`, with a new GPG signing key. The legacy `dl.redpanda.com` repository is scheduled to shut down on *{legacy-repo-shutdown-date}*. If you currently install or upgrade Redpanda from `dl.redpanda.com`, re-point your package manager before then. For steps, including a manual configuration option for environments that cannot pipe a script to `bash`, see xref:migrate:linux-package-repository.adoc[Migrate the Linux Package Repository].
====
