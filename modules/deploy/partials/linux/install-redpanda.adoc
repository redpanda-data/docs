= Install Redpanda

Install Redpanda on each system you want to be part of your cluster. There are binaries available for Fedora/RedHat or Debian systems.

[tabs]
=====
Fedora/RedHat::
+
--
[,bash]
----
curl -1sLf 'https://linux.pkg.redpanda.com/setup-redpanda.rpm.sh' | \
sudo -E bash && sudo yum install redpanda -y
----

--
Debian/Ubuntu::
+
--
[,bash]
----
curl -1sLf 'https://linux.pkg.redpanda.com/setup-redpanda.deb.sh' | \
sudo -E bash && sudo apt install redpanda -y
----
