= rpk
:description: pass:q[The `rpk` is Redpanda's command line interface (CLI) utility.]
:page-aliases: reference:rpk-commands.adoc

`rpk` is a command line interface (CLI) toolbox that let you configure, manage, and tune Redpanda clusters. They also let you manage topics, groups, and access control lists (ACLs). 

`rpk`stands for Redpanda Keeper.

== rpk

`rpk`` is the Redpanda CLI toolbox.

=== Usage

[,bash]
----
rpk [command]
----

=== Flags

[cols="1m,1a,2a"]
|===
|*Value* |*Type* |*Description*
|-h, --help |- |Help for `rpk``.
|-v, --verbose |- |Enable verbose logging (default `false`).
|===
