=== a2a_message processor
:term-name: a2a_message processor
:hover-text: A Redpanda Connect processor that enables pipelines to invoke AI agents using the Agent2Agent protocol.
:category: Redpanda Connect

The `a2a_message` processor allows Redpanda Connect pipelines to send messages to AI agents and receive responses. This enables pipeline-initiated agent interactions where streaming data can trigger agent reasoning and tool execution as part of data processing workflows.

See also: glossterm:processor[], glossterm:a2a-protocol[], glossterm:ai-agent[], glossterm:redpanda-connect[]
