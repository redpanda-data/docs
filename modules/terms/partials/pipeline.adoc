=== pipeline
:term-name: pipeline
:hover-text: A single configuration file running in Redpanda Connect with an input connector, an output connector, and optional processors in between. A pipeline typically streams data into Redpanda from an operational source (for example, `postgres_cdc`) or streams data out of Redpanda into an analytical system (for example, `snowflake_streaming`).
:category: Redpanda Cloud
