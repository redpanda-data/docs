=== local MCP server
:term-name: local MCP server
:hover-text: An MCP server running in a local development environment that connects AI agents to tools and resources on the developer's machine.
:category: Agentic Data Plane

Local MCP servers enable developers to test and debug AI agent integrations on their own machines before deploying to production. They provide the same MCP interface as remote servers but run locally with access to local files, databases, and development resources.

See also: glossterm:mcp-server[], glossterm:remote-mcp[], glossterm:mcp[]
