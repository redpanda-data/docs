== Iceberg: Health monitoring

Redpanda now provides an Admin API endpoint to monitor the health of Iceberg topics, including external catalog reachability and per-partition commit lag. Use it to detect an unreachable catalog and identify the partitions with the highest commit lag.

See xref:manage:iceberg/monitor-iceberg-health.adoc[] for details.
